# API de Barbearia

Bem-vindo ao meu primeiro projeto Back-end em C#!
Este sistema foi construído para gerenciar os agendamentos de uma barbearia, permitindo criar, listar e cancelar horários.

## Tecnologias Utilizadas
* **C# e .NET** (A linguagem e a fábrica)
* **Entity Framework Core** (O gerente do banco de dados)
* **SQLite** (O banco de dados)
* **xUnit** (Testes unitários)
* **Swagger** (Interface de testes da API)
* **HTML/JS** (Front-end simples para visualização)

## O que eu aprendi neste projeto?
- Como criar uma API RESTful do zero.
- Como configurar e salvar dados em um banco de dados real.
- Como criar rotas assíncronas (`async/await`).
- Como escrever Teste Unitários e usar o padrão AAA.
- Como debugar a aplicação e gerar Logs de observalidade.
- Como versionar meu código no GitHub.
